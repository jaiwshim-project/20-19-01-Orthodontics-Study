## 고객 제안 기반 아이디어

### 병원장들이 자주 요청하는 기능

#### 1. "환자 비포/애프터 자동 생성"
- [ ] 초진 사진 + 완료 사진 → 자동 비포/애프터 이미지 생성
- [ ] SNS 마케팅용 자동 편집
- **우선순위**: 높음 (비용 0, 효과 높음)

#### 2. "치료 비용 투명 계산기"
- [ ] 부정교합 종류별 표준 비용 제시
- [ ] 시간에 따른 비용 변화 계산
- [ ] 보험 청구 자동화
- **우선순위**: 중간

#### 3. "직원 교육 자동 채점"
- [ ] 신입에게 케이스 제시
- [ ] 신입이 진단하면 AI가 채점
- [ ] 개선 영역 자동 피드백
- **우선순위**: 중간

#### 4. "환자 만족도 자동 설문"
- [ ] 치료 완료 후 자동 설문 발송
- [ ] NPS 스코어 자동 계산
- [ ] 부정 리뷰 조기 감지
- **우선순위**: 높음
