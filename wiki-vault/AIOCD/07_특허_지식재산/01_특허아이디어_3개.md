# AIOCD 핵심 특허 아이디어 (3개)

## 특허 1️⃣: "AI 기반 교정진단 자동화 시스템"

### 1. 발명의 명칭
AI 기반 교정진단 자동화 시스템 및 그 방법

### 2. 기술 분야
치과 교정학 분야의 AI 기반 진단 자동화 시스템

### 3. 해결하려는 문제
기존 교정진단은 의사의 경험과 직관에 의존하여, 다음과 같은 문제 발생:
- 진단 시간이 길어서 환자 처리 능력 제한
- 진단 편차 발생 (의사마다 다른 진단)
- 신입 의사 교육에 오랜 시간 소요

### 4. 기존 기술의 한계
- WebCeph: 세팔로 측정만 자동화 (진단 자동화 없음)
- Invisalign: 특정 장치에만 적용
- 일반 교과서: 개별 환자 특수성 미반영

### 5. 발명의 핵심 구성

#### 모듈 1: 이미지 분석 모듈
- 치과 사진 입력
- CNN 기반 부정교합 분류 (Angle Class I/II/III)
- 치열궁 형태 분석

#### 모듈 2: 세팔로메트리 분석 모듈
- 3D 또는 2D 세팔로 데이터 입력
- 자동 각도 추출 (SNA, SNB, ANB, etc)
- 정상범위와 비교 분석
- 편차도 계산

#### 모듈 3: 온톨로지 기반 규칙 엔진
- 부정교합 + 세팔로 분석 → 진단 규칙 매핑
- 한국 교정학 표준 기준 적용
- 의료 법규 자동 검증

#### 모듈 4: 치료계획 생성 엔진
- 진단 결과 → 치료 옵션 제시 (3~5개)
- 예상 치료 기간 계산
- 예상 비용 산출

### 6. 작동 흐름

```
입력 (이미지 + 세팔로)
    ↓
이미지 분석 → 부정교합 분류
    ↓
세팔로 분석 → 각도 추출 + 편차 계산
    ↓
온톨로지 매핑 → 진단 규칙 적용
    ↓
치료계획 생성 → 3~5개 옵션
    ↓
의사 검증 → 최종 진단 확정
    ↓
출력 (진단 리포트 + 치료계획)
```

### 7. 기술적 효과
1. **진단 시간 단축**: 20분 → 3분 (85% 감소)
2. **진단 일관성**: 의사마다 다른 진단 제거 (일관성 100%)
3. **정량화된 근거**: 객관적 수치 기반 진단 (신뢰도 향상)
4. **의료사고 감소**: AI 검증으로 오진 방지

### 8. 실시 예
```
예시: 초신 환자 (26세 여성, Angle Class II)

입력:
- 정면 구강사진 3장
- 측면 사진 1장
- 2D 세팔로 사진

AI 처리 (3분):
- 부정교합: Angle Class II Division 1로 분류
- 세팔로: ANB 각도 8.5° (정상범위 2~4°)
- 평가: 상악전돌 (Maxillary prognathism)

치료계획 제시:
- 옵션 1: 고정 교정장치 (24개월)
- 옵션 2: 투명 교정장치 (28개월)
- 옵션 3: 수술 동반 교정 (36개월) - 전문가 검토 필요

의사 결정:
- 환자가 원하는 장치 선택
- 예상 결과 3D 시뮬레이션 제시
```

### 9. 청구항 후보

1. **청구항 1 (독립항)**: 
컴퓨터 기반 교정진단 시스템으로, 이미지 분석 모듈, 세팔로메트리 분석 모듈, 온톨로지 기반 규칙 엔진, 치료계획 생성 엔진으로 구성되며, 입력된 치과 이미지와 세팔로 데이터로부터 자동으로 부정교합을 분류하고 교정 치료계획을 생성하는 시스템.

2. **청구항 2**: 청구항 1의 시스템의 동작 방법

3. **청구항 3**: 이미지 분석 모듈의 구체적 구성 (CNN 기반)

4. **청구항 4**: 세팔로 분석 모듈의 자동 각도 추출 방법

5. **청구항 5**: 온톨로지 기반 규칙 엔진의 진단 매핑 방법

## 특허 2️⃣: "치아 이동 3D 시뮬레이션 및 임상가이드 자동 생성 방법"

### 1. 발명의 명칭
치아 이동 3D 시뮬레이션 및 단계별 임상가이드 자동 생성 시스템

### 2. 해결 문제
- 환자는 진단을 이해하지 못함 → 신뢰도 낮음
- 의사는 임상가이드 작성에 시간 소비
- 신입 의사는 각 단계별 지침이 없어서 오류 발생 위험

### 3. 핵심 구성

#### 모듈 1: 3D 모델 생성
- 초기 CT/3D 스캔 데이터 입력
- 3D 모델 자동 생성 (Three.js)

#### 모듈 2: 시뮬레이션 엔진
- 치료계획 입력 (와이어 크기, 고무줄, 힘 방향 등)
- 생물역학 계산 (치아 이동량 계산)
- 월별/분기별 위치 자동 계산

#### 모듈 3: 임상가이드 생성
- 각 단계별 시술 내용 자동 작성
- 와이어 사이즈 추천
- 고무줄 강도 추천
- 다음 내방 일정 추천

### 4. 기술적 효과
1. **환자 신뢰도**: 3D 시뮬레이션으로 결과를 미리 보여줌 (신뢰도 +25%)
2. **의료 사고 감소**: 단계별 가이드로 오류 방지
3. **업무 효율**: 의사의 가이드 작성 시간 제거

### 5. 청구항 예시
- 초기 3D 모델 + 치료계획 → 단계별 위치 계산 → 3D 동영상 생성
- 동영상 + AI 분석 → 임상가이드 자동 작성
- 가이드 + 의사 검증 → 최종 임상 프로토콜 확정

## 특허 3️⃣: "AI 진단 기반 외국인 환자 상담 자동화 시스템"

### 1. 발명의 명칭
AI 교정진단 결과 기반 다국어 상담통역 및 진료설명 자동화 시스템

### 2. 결합 플랫폼
- AIOCD (진단 자동화)
- MedVo (다국어 상담통역)
- Insight Bank Hub (신뢰 콘텐츠)

### 3. 해결 문제
- 외국인 환자는 언어 장벽으로 인한 소통 어려움
- 병원은 외국인 환자 응대 인력 부족
- 진료설명의 의료 정확성 담보 어려움

### 4. 핵심 기술 흐름

```
AIOCD 진단 결과 (한글)
    ↓
자동 번역 (30개 언어)
    ↓
MedVo 다국어 상담
    ↓
진료설명 스크립트 생성
    ↓
AI 상담코칭
    ↓
예약 확정
```

### 5. 기술적 효과
1. **시장 확대**: 외국인 환자 3배 증가
2. **비용 절감**: 통역사 채용 비용 제거
3. **의료 정확성**: AI 기반 정확한 진료설명

### 6. 청구항 예시
- 교정진단 결과 자동 다국어 변환 방법
- 변환된 내용 기반 다국어 상담 시스템
- 상담 데이터를 한글 콘텐츠로 역변환하는 피드백 방법
