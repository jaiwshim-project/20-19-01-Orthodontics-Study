# 핵심 기능 목록

## 1. 필수 기능 (P0 - 반드시 필요)

| 기능 | 설명 | 사용자 |
|---|---|---|
| **AI 교정진단** | 치과 이미지 입력 → AI 자동 부정교합 분류 | 교정의 |
| **세팔로 데이터 입력** | 3D 또는 2D 세팔로메트리 값 입력 | 교정의 |
| **자동 측정** | WebCeph 연동으로 각도·거리 자동 계산 | 교정의 |
| **진단 리포트** | 진단 결과 + 수치 기반 분석 리포트 생성 | 교정의 |
| **환자 교육 콘텐츠** | AI가 진단 결과 기반 환자 설명 자료 생성 | 환자·교정의 |
| **기본 로그인** | 병원별·개인별 계정 분리 | 모두 |
| **데이터 저장** | 환자 케이스 저장 및 검색 | 교정의 |

---

## 2. 차별화 기능 (P1 - 사업화에 중요)

| 기능 | 설명 | 기대 효과 |
|---|---|---|
| **3D 치아이동 시뮬레이션** | AI 기반 치아 이동 예측 (6개월 후, 1년 후, 최종) | 환자 신뢰도 ↑ (이미지는 1000 단어) |
| **AI 치료계획 생성** | 진단 → 교정 장치 추천 → 단계별 가이드 | 신입 의사도 전문의 수준 진단 |
| **교정의 임상가이드** | 단계별 지침 (와이어 사이즈, 고무줄 강도, 방문 간격) | 진료 시간 70% 단축 |
| **교정의 학습 플랫폼** | 1000개 케이스 기반 AI 학습 (유사 케이스 추천) | 신입·경력 의사 교육 |
| **케이스 라이브러리** | 성공 사례 DB + 비교 분석 기능 | 임상 결정 지원 |
| **병원 관리 대시보드** | 월간 진단 수, 환자 만족도, 매출 추이 | 경영진 의사결정 |
| **환자 앱 연동** | 환자가 진단 결과·진행 상황 확인 | 진료 만족도 ↑ |

## 4. AI 기능 (P1-P2)

| AI 엔진 | 입력 | 출력 | 상태 |
|---|---|---|---|
| **부정교합 분류** | 치과 이미지 | 치료 난이도, 권장 장치 | ✓ |
| **세팔로 분석** | 3D/2D 세팔로 데이터 | 각도 평가, 정상 범위와의 편차 | ✓ |
| **치료계획 추천** | 진단 결과 | 예상 기간, 비용, 장치 순서 | 🔄 |
| **시뮬레이션** | 초기 모델 + 치료계획 | 3D 동영상 (6개월/1년/최종) | 🔄 |
| **유사 케이스 검색** | 현재 케이스 | 성공 사례 3~5개 (추천) | ⏳ |
| **예후 예측** | 진단 + 환자 협력도 | 예상 결과 + 위험 요소 | ⏳ |
