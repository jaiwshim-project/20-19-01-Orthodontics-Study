# 플랫폼 한 줄 정의

## 기본 정의

AIOCD(AI Orthodontic Comprehensive Diagnosis)는 치과 이미지와 세팔로메트리 데이터를 입력받으면, AI가 자동으로 **교정진단, 3D 치아이동 시뮬레이션, 맞춤형 치료계획**을 제시하고, **교정의 임상가이드**와 **환자 교육 콘텐츠**를 함께 생성하여, 교정의의 진단 시간을 70% 단축하고 환자 만족도를 높이는 **디지털 교정학 플랫폼**이다.

---

## 30초 설명

AIOCD는 교정의의 "진단 비서"입니다.

이미지를 올리면 AI가 자동으로:
- 교정진단 (부정교합 분류)
- 치료계획 제시
- 3D 시뮬레이션 (치아 이동 예측)
- 환자 교육 자료 생성
- 임상가이드 (단계별 지침)

을 3분 안에 만들어줍니다.

---

## 3분 설명

### 현재 교정의의 문제
- 초진 환자 진단에 **20~30분**이 걸린다
- 세팔로메트리 분석을 **수동으로** 한다
- 환자 설명 자료를 **반복해서** 만든다
- 치료계획을 **경험과 직관**에 의존한다

### AIOCD가 제공하는 가치
1. **진단 자동화**: 20분 → 3분
2. **정량화된 치료계획**: 수동 분석 → AI 기반 설계
3. **환자 설득**: 자동 생성 콘텐츠 (교정 기간, 비용, 결과)
4. **임상가이드**: 각 단계별 조기 지침 (언제 와이어 변경, 언제 고무줄 조정)
5. **학습 플랫폼**: 교정의·직원·환자 교육

---

## 고객에게 설명하는 문장

> "교정진단을 받으실 때 의료진이 AI와 함께 정확한 진단과 3D 치아 이동 예측을 보여드리고, 치료 기간과 비용도 명확하게 안내해드립니다. 투명한 진료 과정은 환자분의 신뢰를 높입니다."

---

## 투자자에게 설명하는 문장

> "AIOCD는 한국의 1만 5천여 교정의를 대상으로, 진단 자동화 × AI 교육 × 환자 교육을 결합한 SaaS 플랫폼입니다. 월 500만 원 × 1000개 치과 = 연 60억 원 매출 잠재력을 가진 B2B 의료 솔루션입니다. 추가로 환자 애플리케이션(B2C)으로도 확장 가능합니다."

| 대상 | 메시지 |

| 교정의 | 진단 시간 70% 단축, 정확성은 190% 향상 |
| 병원장 | 환자 만족도 ↑, 진료 효율성 ↑, 재계약율 ↑ |
| 환자 | 투명한 진료, 예측 가능한 결과, 신뢰 형성 |
| 기자·투자자 | 한국 교정학의 디지털 혁신 |

---

## 기술적 정의

AIOCD = 
- **입력**: 치과 이미지 + 세팔로메트리 데이터
- **처리**: Computer Vision + Deep Learning + 온톨로지 기반 진단 엔진
- **출력**: 진단 리포트 + 3D 시뮬레이션 + 치료계획 + 환자 교육 자료 + 임상가이드

---

## 차별화 요약

| 항목 | 기존 방식 | AIOCD |
|---|---|---|
| 진단 시간 | 20~30분 | 3분 |
| 진단 방식 | 의사의 경험·직관 | AI 기반 정량화 |
| 환자 교육 | 일반적 설명 | AI 생성 맞춤 콘텐츠 |
| 시뮬레이션 | 없음 | 3D 치아 이동 예측 |
| 임상가이드 | 의사의 기억 | AI 단계별 자동 가이드 |

---

**최종 정의 공식**

```
AIOCD 
= 치과 의료 데이터 × AI 교정학 지식 × 자동화 엔진
= 교정의의 진단 효율성 + 환자 만족도
= 한국 교정학의 디지털 변환
```
