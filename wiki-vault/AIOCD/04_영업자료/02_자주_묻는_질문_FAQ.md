# FAQ: AIOCD에 대해 자주 묻는 질문

## Q1. WebCeph나 다른 세팔로 분석 프로그램과 뭐가 다른가요?

**A**: 
- **WebCeph**: 세팔로메트리 **측정**만 자동화 (각도 계산)
- **AIOCD**: **진단**부터 **계획**까지 전체 자동화

```
WebCeph: 측정 자동화 (시간 절약 3분)
AIOCD:   진단 + 계획 + 시뮬 + 교육 (시간 절약 17분 + 신뢰도 향상)
```

## Q2. AI가 정말 정확한 진단을 할까요? 의사의 판단이 더 중요하지 않나요?

**A**: 
AI와 의사의 역할이 다릅니다.

```
AI의 역할: 객관적 기준 제시 (정량화된 진단)
의사의 역할: AI 결과를 검증하고 환자 특수성 고려

예: 
- AI가 "Angle Class II" 제시
- 의사가 "이 환자는 수술이 필요함" 또는 "보존 치료 가능" 판단

→ 의사의 경험은 더 높은 수준의 의사결정에 집중
```

## Q13. 치료계획은 정말 AI가 자동으로 만드나요? 너무 위험하지 않나요?

**A**: 
**AI가 제시하고, 의사가 최종 결정**합니다.

```
AI의 역할: 1,000개 사례 기반 "추천"
의사의 역할: "이 환자에게는 맞나?" 검증 + 최종 결정

예:
- AI가 추천: "와이어 크기 0.016inch"
- 의사가 판단: "이 환자는 민감하니 0.014inch로 시작"

→ 의사의 최종 책임
```
