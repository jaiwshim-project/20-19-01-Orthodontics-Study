# 📸 Kim의 강의 슬라이드 가이드 (45장)

## 📌 개요

"교정학 자료-김용을 원장" 폴더의 45개 슬라이드 이미지(JPG)는 Kim Yong Eul의 실제 강의 자료입니다.

각 슬라이드는 명문출판사의 11개 챕터와 체계적으로 매핑되어 있어, **슬라이드 → 해당 교과서 챕터**로 깊이 있는 학습이 가능합니다.

## 💡 사용 방법

### 방법 1: 슬라이드 → 교과서로 이동
1. **슬라이드 1-5** 보기
2. **Chapter 1** (현대 교정학의 발전과 과제.docx) 읽기
3. 해당 개념 깊이 있게 학습

### 방법 2: 교과서 → 슬라이드로 복습
1. **Chapter X** 읽기
2. 해당 범위 **슬라이드 XX-YY** 이미지 보면서 요약
3. 핵심 개념 정리

### 방법 3: learning-platform.html 사용
- 📚 **자료(Resources)** 탭에서 슬라이드 필터링
- 슬라이드 그룹별로 해당 교과서 추천
- 순차적 학습 경로 안내
