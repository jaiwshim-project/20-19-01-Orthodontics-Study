## 🔗 사용 방법 3가지

### 방법 1: 슬라이드 → 교과서 (권장)
```
1. learning-platform.html 열기
   ↓
2. 자료 탭 → JPG 필터링
   ↓
3. 원하는 슬라이드 범위 선택
   (예: 슬라이드6-10 선택)
   ↓
4. 해당 챕터 이동
   (Chapter 2: Growth and Development 읽기)
   ↓
5. 슬라이드와 교과서 함께 학습
```

### 방법 2: 교과서 → 슬라이드 (복습)
```
1. Chapter X 읽기
   (예: Chapter 5 읽음)
   ↓
2. 슬라이드-가이드.md에서 해당 범위 확인
   (Chapter 5 = 슬라이드 22-26)
   ↓
3. 슬라이드 이미지 보면서 요약 정리
```

### 방법 3: Wiki 활용
```
1. wiki-vault 열기 (Obsidian)
   ↓
2. 슬라이드-가이드.md 클릭
   ↓
3. 원하는 슬라이드 범위 선택
   ↓
4. 관련 자료 링크로 이동
   (클릭 → 교과서/참고 자료 열림)
```
