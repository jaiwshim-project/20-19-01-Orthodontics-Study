# 🔧 Webceph & Cephalometric 교정학 방법론

## ⚠️ 중요 공지

**이 문서는 Kim Yong Eul의 Equilibrium Zone 방법론과 별개의 교정학 접근법을 설명합니다.**

---

## 📌 Webceph 방법론

### 정의
```
X레이 측면(lateral) 촬영을 기반으로 앞니의 
돌출도(protrusion)와 기울기(inclination)를 분석하여
교정 치료를 계획하는 방식
```

### Webceph 플랫폼 실제 기능

**공식 플랫폼**: https://webceph.com/ko/cohort/

#### 1️⃣ 분석 자료
- **X레이**: 측면(lateral) 촬영 (단일 2차원 이미지)
- **사진**: 정면 사진 (참고용)
- **플랫폼 제공**: Web-based + Mobile App (Android/iOS)

#### 2️⃣ 핵심 기능

**자동 세팔로 트레이싱**
```
X레이 이미지 업로드
    ↓
AI가 자동으로 해부학적 랜드마크 감지
    ↓
자동 트레이싱 (랜드마크 연결)
    ↓
계측값 자동 계산
```

**맞춤형 분석 및 계측**
```
기본 제공 계측법
    ↓
"분석법 위자드" - 나만의 분석법 생성
"계측치 위자드" - 나만의 계측 기준 정의
    ↓
임상 증례를 템플릿 형태로 저장
```

**영상 처리 및 비교**
```
세팔로 영상 자동 중첩 → 치료 전후 비교
싱글/듀얼/템플릿 세 가지 이미지 뷰어 모드
치료 결과를 이미지/동영상으로 저장
```

#### 3️⃣ 측면 X레이 기반 진단
```
앞니의 위치 분석:
├─ 앞으로 돌출됨 (Protrusion)
│  └─ 어느 정도로 앞으로 나왔는가?
├─ 안으로 기울어짐 (Inclination)
│  └─ 어느 정도로 기울어졌는가?
└─ 수직 위치 (Vertical position)
   └─ 위/아래 어디에 위치하는가?
```

#### 4️⃣ 치료 계획 수립 방식
- 측면 X레이에서 자동 계측 (automatic measurement)
- 이상적인 돌출도/기울기 설정 (standard values or custom)
- 부족한 부분을 교정으로 보정
- **Equilibrium Zone 개념 없음**
- **공간분석 없음** (2차원 측면 분석만)

### 장점
- ✅ 측면 미적 분석 용이
- ✅ X레이 기반 객관적 수치화
- ✅ 전통적 교정학 교육과 호환
- ✅ 임상 현장에서 오랫동안 사용됨

### 단점
- ❌ 측면 분석만 가능 (앞뒤 관점만)
- ❌ 공간 부족/초과 원인 분석 부족
- ❌ 3차원 입체 분석 불가
- ❌ 개인별 치아배열 특성 미반영
- ❌ 재발률 상대적으로 높음

## 📌 Cephalometric 분석

### 정의
```
두개골과 악골의 형태, 크기, 위치 관계를
X레이에서 계측하여 교정 진단을 하는 방식
```

### 특징

#### 1️⃣ 주요 계측값 (예시)
```
골격 관계:
├─ ANB (상악과 하악의 상대적 위치)
├─ SNA (상악 위치)
├─ SNB (하악 위치)
└─ Facial plane angle (안면각)

치아 위치:
├─ Overjet (앞뒤 거리)
├─ Overbite (위아래 거리)
└─ Incisor inclination (앞니 기울기)
```

#### 2️⃣ 분석 방식
- 두개골 뼈의 형태 분석
- 악골 성장 패턴 평가
- 치아와 뼈의 관계 계측
- 표준값과 비교하여 이상 판단

#### 3️⃣ 치료 계획 수립
- 뼈 성장 예측
- 성장기/성인 판단
- 발치 여부 결정 (세팔로 값 기반)
- 외과적 수술 필요성 판단

## 🔄 Kim vs. Webceph vs. Cephalometric 비교

### 핵심 차이점

| 구분 | **Kim (Equilibrium Zone)** | **Webceph** | **Cephalometric** |
|------|--------------------------|-----------|------------------|
| **분석 범위** | 5방향 (정/좌/우측면/상/하부) | 측면 X레이 | 측면 X레이 + 계측 |
| **3차원 분석** | ✅ 예 | ❌ 아니오 (2차원) | ❌ 아니오 (2차원) |
| **핵심 개념** | Equilibrium Zone | Protrusion/Inclination | Skeletal/Dental relationship |
| **공간분석** | ✅ 공간 부족/초과 | ❌ 없음 | △ 간접적 |
| **진단 중심** | 공간 부족/초과 원인 | 앞니 돌출도 | 골격 성장 패턴 |
| **AI 플랫폼** | 20-19 Orthodontics AI | Webceph AI | Cephalo AI (일반) |
| **개인화 수준** | 높음 (개인 공간분석) | 중간 | 중간 |
| **재발률** | 낮음 | 상대적 높음 | 중간~높음 |

## 📊 진단 흐름 비교

### Kim Yong Eul 방법론
```
5방향 촬영 (정/좌/우측면/상/하부)
    ↓
5방향 영상 분석 (입체 좌표 추출)
    ↓
Equilibrium Zone 인식 (AI 자동화)
    ↓
공간분석 (공간 부족/초과 정량화)
    ↓
부정교합 유형 판단
    ↓
최적 교정 계획 수립
```

### Webceph 방법론
```
측면 X레이 촬영 (+ 정면 사진)
    ↓
측면 분석 (2차원 계측)
    ↓
앞니 돌출도 & 기울기 평가
    ↓
이상적 위치와의 거리 계산
    ↓
교정으로 수정할 양 결정
    ↓
치료 계획 수립
```

### Traditional Cephalometric
```
측면 X레이 촬영
    ↓
두개골 및 악골 계측
    ↓
표준값과의 비교
    ↓
성장 패턴 평가
    ↓
발치/비발치 결정
    ↓
치료 계획 수립
```

## 🎯 선택 기준

### Kim 방법론 사용 시기
- 장기 안정성이 중요한 경우
- 공간 부족 (crowding) 원인을 정확히 파악 필요
- 개인별 맞춤 치료 계획 필요
- 재발 방지가 우선순위

### Webceph 사용 시기
- 측면 미적분석 중심 치료
- X레이 기반 빠른 진단 필요
- 전통적 세팔로 교육 배경
- 간단한 돌출 케이스

### Cephalometric 사용 시기
- 골격 성장 패턴 분석 필요
- 성장기 환자 평가
- 외과적 수술 필요성 판단
- 성장 예측 필요
