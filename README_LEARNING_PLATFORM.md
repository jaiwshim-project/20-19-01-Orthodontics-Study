## 💡 핵심 개념 미리보기

### 교정학의 4가지 기둥

```
1. 정확한 진단 (Diagnosis)
   └─ X선, 사진, 모형 분석 → 부정교합 분류

2. 과학적 계획 (Planning)
   └─ 최종 목표 설정 → 치료 순서 결정

3. 효과적 실행 (Execution)
   └─ 장치 선택 → 힘 조절 → 위치 이동

4. 체계적 평가 (Evaluation)
   └─ 결과 측정 → 재발 예방 → 유지
```

### 4대 AI 엔진 (플랫폼이 제공하는 것)

```
① Extraction AI (발치 판단)
   입력: X선, 사진, 공간분석
   출력: 발치 여부 + 근거

② Growth Prediction (성장 예측)
   입력: CVMS stage, 세파로 값, 나이
   출력: 예상 성장 방향 + 타이밍

③ Facial Simulation (안모 시뮬레이션)
   입력: 현재 프로파일 + 치료 변수
   출력: 예상 안모 (측면)

④ Recurrence Prediction (재발 예측)
   입력: 부정교합 유형, 나이, 성별
   출력: 1·3·5·10년 재발 확률
```
