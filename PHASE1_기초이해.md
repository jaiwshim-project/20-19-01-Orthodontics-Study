## 💡 개념 정리 (필독!)

### 1. 부정교합의 분류

**Angle 분류** (가장 기본)
```
정상 교합     : 상 제1대구치와 하 제1대구치가 특정 관계
Class I       : 상악골 vs 하악골 크기는 정상, 치아만 문제
Class II      : 하악골이 상악골에 비해 뒤에 있음
              Class II Division 1: 상 중절치가 앞으로 튀어나감
              Class II Division 2: 상 중절치가 안쪽으로 들어감
Class III     : 하악골이 상악골에 비해 앞으로 나옴
```

**수직 분류** (위아래 관계)
```
정상         : 수직 중첩(overbite) 2-3mm
Deep Bite    : 수직 중첩이 깊음 (심각한 경우 4mm+)
Open Bite    : 앞니가 물리지 않음 (음수 overbite)
```

**전후 분류** (앞뒤 관계)
```
정상         : 전방 중첩(overjet) 2-3mm
Protrusion   : 앞니가 앞으로 튀어나감
Retrusion    : 앞니가 뒤로 들어감
```

### 2. 성장과 발육

**골격 성장 패턴** (3가지)
```
Horizontal Pattern  : 앞뒤로 성장 (정상)
                     → 발치 고려 가능

Vertical Pattern    : 수직으로 성장 (길게 성장)
                     → 보통 비발치

Mixed Pattern       : 여러 방향으로 복합 성장
```

**CVMS 성장기 판정** (Cervical Vertebral Maturation Stage)
```
Stage 1-2  : 성장 활발 (어린이, 9-10세)
Stage 3-4  : 성장 진행 중 (초기 사춘기)
Stage 5-6  : 성장 완료 (후기 사춘기, 16-17세)
Stage 7-8  : 성장 완료 (성인, 18세+)

→ 성장 시기를 알아야 성장 방향 조정 가능!
```

### 4. 혼합치열기 공간분석

**혼합치열기란?** 
유치와 영구치가 섞여 있는 시기 (약 6-12세)

**공간분석의 목적**:
"현재 공간이 충분한가? → 발치가 필요한가?" 판단

**분석 방법**:
```
Step 1: 상악 영구 전치 크기 측정 (4개 중절치 + 4개 견치)
Step 2: 상악 유 견치 & 제1대구치 크기 (제거될 부분)
Step 3: 공간 부족도 = 측정값1 - 측정값2 - 이용가능 공간

결과:
- 공간 충분  : 비발치 가능
- 공간 부족  : 발치 고려
```

### 5. 세파로메트릭 분석의 3가지 기본값

```
① SNA: 상악골 위치 (정상: 80-82°)
   - 증가 = 상악이 앞으로
   - 감소 = 상악이 뒤로

② SNB: 하악골 위치 (정상: 78-80°)
   - 증가 = 하악이 앞으로
   - 감소 = 하악이 뒤로

③ ANB: 상악과 하악의 상대 관계 (정상: 2-4°)
   - 증가 = Class II (하악이 뒤)
   - 감소 = Class III (하악이 앞)

→ 이 3가지로 골격 부정교합 판정 가능!
```
