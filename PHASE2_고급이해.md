## 💡 개념 정리 (필독!)

### 1. Extraction AI의 핵심 로직

```
혼합치열기 (6-12세)에서 영구치 공간 부족 판단

Step 1: 상악 전치 크기 측정
└─ 절치 4개 + 견치 2개의 mesiodistal diameter 합계

Step 2: 현재 유치 공간 측정
└─ 유 견치 + 유 제1대구치 (제거될 공간)

Step 3: 이용 가능 공간 계산
└─ 원숭이 공간 (primate space)
└─ 발육 여유 (developmental space)

Step 4: 공간 부족도 판정
└─ 부족도 < 2mm : 비발치 가능
└─ 부족도 > 4mm : 발치 필요
└─ 2-4mm 사이 : 성장 관찰 필요
```

**플랫폼 적용**:
- X선과 치모형 스캔에서 자동 측정
- AI가 계산식 실행
- 치과의사 확인/수정 가능

---

### 2. Growth Prediction의 핵심

```
성장 예측 = CVMS stage + 세파로 계측값 + 나이/성별

현재 CVMS stage 3 (초기 사춘기)
→ 예상 성장 기간: 2-3년
→ 성장량: 개인차 크지만 약 3-5mm

Ricketts' analysis로 성장 방향 판정
└─ Y-axis angle (낮음 = horizontal, 높음 = vertical)
└─ Frankfort-mandibular plane angle

수직 성장 패턴 (FMA > 30°)
→ 비발치 + 수직 제어 필요
```

**플랫폼 적용**:
- CVMS 자동 분류 (AI의 측면 두부 X선 분석)
- 세파로 계측값 자동 계산
- 성장 시뮬레이션 제공

---

### 3. Facial Simulation의 핵심

```
치료 변수 변경 → 안모 변화 예측

변수 1: 절치 위치 (수평/수직)
├─ 상 절치 전진 1mm → 상 입술 돌출 0.8mm
├─ 하 절치 후퇴 1mm → 하 입술 함몰 0.5mm
└─ 수직 위치 ↑ → Incisor display ↑

변수 2: 상악 위치
└─ 상악 forward movement 1mm → profile ↑ convex

변수 3: 하악 위치
└─ 하악 전진 1mm → profile ↓ convex, chin projection ↑

변수 4: 골격 성장
└─ 성장 예측값 반영

선형 회귀 모델로 계산
└─ 각 변수별 영향도 weight 적용
```

**플랫폼 적용**:
- 3D 안모 스캔 입력
- 치료 변수 시뮬레이션
- 최종 예상 안모 3D 이미지 출력

### 4. Recurrence Prediction의 핵심

```
재발 위험도 점수 = f(부정교합유형, 나이, 유지방법, 골격)

부정교합별 기본 재발률
├─ Class I malocclusion: 10-20% (낮음)
├─ Class II Div 1: 30-40% (중간)
├─ Class III: 20-30% (중간)
└─ Open bite: 70-80% (매우 높음)

나이 인자
└─ 어릴수록 (골개조 활발) → +10-20% 가산

유지 방법 인자
├─ Fixed retainer 24/7: -10% (감소)
├─ Removable (매일): -5% (약간 감소)
└─ Removable (간헐적): ±0% (기준)

골격 인자
├─ Horizontal pattern: -10% (감소)
├─ Vertical pattern: +20% (증가)
└─ Normal: ±0% (기준)

최종 재발 확률 = 기본율 + 나이인자 + 유지인자 + 골격인자
```

**플랫폼 적용**:
- 환자 정보 입력 → 재발 위험도 자동 계산
- 1년, 3년, 5년, 10년 누적 재발률 제시
- 유지 방법별 재발률 비교
