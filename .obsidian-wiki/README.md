## 🎯 사용 방법

### 플랫폼과의 연동
1. **웹 플랫폼** (HTML) → 학습 자료 접근 + 진도 추적
2. **Obsidian Vault** (마크다운) → 상세 학습 노트 작성

### 학습 노트 작성
- 플랫폼의 **📊 진도 페이지**에서 "🧠 Obsidian으로 노트 작성" 버튼 클릭
- 해당 Phase 폴더 내 `학습노트_Template.md` 템플릿 사용
- 학습한 내용을 자신의 언어로 정리

### 진도 동기화
- 플랫폼: Phase 진도율 입력 (0-100%)
- JSON 데이터 export → 백업/분석
- Obsidian과 자동 메타데이터 동기화
