# 🏗️ 교정학 학습 플랫폼 아키텍처

## 🎯 플랫폼의 목표

**학습 자료에 있는 모든 임상 지식을**
```
효과적으로 접근하고
↓
단계별로 학습하고
↓
개념으로 정리하고
↓
플랫폼 개선에 적용하기
```

---

## 📊 플랫폼 구조

### 입출력 흐름

```
입력 (학습 자료)
├─ 53개 이미지 (JPG, PNG)
├─ 8개 PPTX (200MB)
├─ 4개 MP4 (영상)
└─ 12개 문서 (PDF, DOCX)
    ↓
┌─────────────────────────────────┐
│  학습 플랫폼 (웹앱)              │
├─────────────────────────────────┤
│ 1. 자료 관리 & 검색              │
│ 2. Phase별 학습 경로             │
│ 3. 개념 정리 도구                │
│ 4. 진도 추적                     │
│ 5. 플랫폼 개선 연동              │
└─────────────────────────────────┘
    ↓
출력 (학습 성과)
├─ 단계별 학습 완료
├─ 개념 정리 노트
├─ 플랫폼 개선 아이디어
└─ 임상 근거 기반 기획 능력
```

---

## 🎨 5가지 핵심 모듈

### 1️⃣ 자료 관리 & 검색 (Resource Hub)

**목표**: 74개 파일을 쉽게 찾고 접근

**기능**:
```
자료 탐색
├─ 카테고리별 (이미지/영상/PPTX/문서)
├─ Phase별 (Phase 1/2/3)
├─ 주제별 (부정교합/진단/AI 엔진)
├─ 파일명 검색
└─ 추천 자료

자료 상세보기
├─ 메타데이터 (크기, 수정일, 타입)
├─ 미리보기 (썸네일)
├─ 태그
├─ 관련 자료 추천
└─ 학습 상태 표시 (미학습/학습중/완료)
```

**UI**:
```
┌─────────────────────────────────┐
│ 🔍 자료 검색                     │
│ [                           ]   │
└─────────────────────────────────┘

┌──────────┬──────────┬──────────┐
│ 이미지   │ PPTX     │ 영상     │
│ 53개     │ 8개      │ 4개      │
├──────────┼──────────┼──────────┤
│ 📸       │ 📊       │ 🎬       │
│ 슬라이드 │ 교정학   │ 교정학   │
│ 1-45     │ 자료.ppt │ 01.mp4   │
└──────────┴──────────┴──────────┘

[자세히 보기] [학습 시작] [태깅]
```

### 2️⃣ Phase별 학습 경로 (Learning Paths)

**목표**: 단계별로 체계적 학습

**구조**:
```
Phase 1: 기초 이해 (1-2주)
├─ Week 1
│  ├─ Day 1: 교정학 01.mp4 (영상 27분)
│  ├─ Day 2-5: 슬라이드 1-7 (이미지)
│  └─ Day 6-7: 웹셉 세팔로.pdf (문서)
│
└─ 학습 목표: 부정교합 분류, CVMS, 진단 도구

Phase 2: 고급 이해 (2-3주)
├─ Week 1
│  ├─ 슬라이드 8-20
│  ├─ The_Bio-Digital_Blueprint.pptx
│  └─ 교정학 02.mp4
│
└─ 학습 목표: 4대 AI 엔진 이론

Phase 3: 플랫폼 적용 (2-3주)
├─ Week 1
│  ├─ Open_Bite_Clinical_Blueprint.pptx
│  ├─ AI_Powered_Precision_Orthodontics.pptx
│  └─ 플랫폼 API 분석
│
└─ 학습 목표: 자주적 기획 능력
```

**UI**:
```
┌────────────────────────────────────┐
│ Phase 1: 기초 이해                  │
│ 1-2주 | 10시간 | 진도 35%          │
├────────────────────────────────────┤
│ ✓ Day 1: 교정학 01.mp4             │
│ ✓ Day 2: 슬라이드 1-3 (완료)       │
│ ◆ Day 3: 슬라이드 4-7 (진행중)     │
│ • Day 4: 슬라이드 8-10 (예정)      │
│ • Day 5: 웹셉 세팔로.pdf (예정)   │
└────────────────────────────────────┘

[이전] [계속 학습] [상세 보기] [다음]
```

### 3️⃣ 개념 정리 도구 (Concept Note)

**목표**: 학습한 내용을 정리하고 이해도 높이기

**기능**:
```
개념 템플릿
├─ 용어 정의 (한 문장)
├─ 핵심 개념 (3-5줄)
├─ 다이어그램/예시
├─ 관련 개념 연결
├─ 플랫폼 연동 (어느 API와 관련?)
└─ 자신의 말로 설명

개념 정리 모음
├─ 부정교합 분류 (Angle 3가지)
├─ CVMS 8단계
├─ 세파로메트릭 분석
├─ 발치 판단 기준 8가지
└─ ... (총 100+ 개념)
```

**UI**:
```
┌─────────────────────────────────┐
│ 📝 개념 정리: Angle 분류          │
│ Phase: 1 | 난이도: ⭐⭐          │
├─────────────────────────────────┤
│ 정의:                            │
│ 상악 제1대구치와 하악 제1대구치의│
│ 관계에 따른 부정교합 분류 체계   │
│                                  │
│ 핵심:                            │
│ Class I   : 정상                 │
│ Class II  : 하악이 뒤(3가지)     │
│ Class III : 하악이 앞            │
│                                  │
│ 다이어그램: [그림 삽입]          │
│                                  │
│ 플랫폼 연동:                      │
│ → extraction-ai.js의 분류 기준   │
│ → diagnose.js에서 사용           │
└─────────────────────────────────┘

[태깅] [공유] [편집] [삭제]

### 5️⃣ 플랫폼 개선 연동 (Application)

**목표**: 학습 → 플랫폼 개선으로 연결

**기능**:
```
학습 내용 → 플랫폼 개선 아이디어

① 발치 판단 로직
   학습 자료: The_Equilibrium_Blueprint.pptx
   ↓
   플랫폼 함수: extractionDecision()
   ↓
   개선 제안: [작성하기]
   - 발치 기준 추가 (예: 골밀도 고려)
   - UI 개선 (시각적 판단 기준)
   - 정확도 평가

② 성장 예측 알고리즘
   학습 자료: Bio-Digital_Blueprint.pptx
   ↓
   플랫폼 함수: growthPrediction()
   ↓
   개선 제안: [작성하기]
   - CVMS 단계 자동 인식
   - 성장 예측 신뢰도 표시
   - 장기 추적 가능성 추가
