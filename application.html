모듈 역할 관련 기술

1. 사용자가 5방향 사진 및 환자 정보 입력
2. 시스템이 사진 분석 및 측정값 추출
3. EZM 알고리즘 실행 (공간분석, 성장예측, 진단)
4. AI가 치료 계획 제안
5. 진단 결과 및 추천 표시

🔍 2단계: 웹셉 분석 모듈 이해

웹셉(Web Cephalometry) 분석의 의미

웹기반 세팔로메트릭 분석은 두개 방사선 사진(Cephalometric X-ray)의 각도와 거리를 측정하여 얼굴의 골격 특징을 분석합니다. 20-19 플랫폼에서는 이를 자동화하여 수동 측정 시간을 단축하고 오류를 줄입니다.

주요 측정 항목

SNA

상악골의 전후방 위치

SNB

하악골의 전후방 위치

ANB

상하악의 상대적 위치

MP-H

하악평면 각도

IMPA

하전치의 경사각

PP-MP

악간 관계각

📊 3단계: EZM 진단 로직 구현

공간분석 알고리즘

공간분석은 다음 단계를 통해 발치 필요 여부를 판단합니다:

1. 필요 공간 계산 = 4개 대구치 폭 + 4개 소구치 폭 + 4개 전치 폭
2. 이용 가능 공간 측정 = 양쪽 제1대구치 사이 호의 길이
3. 공간 불일치도 (CRD) = 필요 공간 - 이용 가능 공간
4. 발치 판단: CRD > 기준값 → 발치 필요

진단 프로세스

Step 1: 환자 데이터 입력 (나이, 성별, 주소 등)
Step 2: 5방향 사진 분석 (정면, 측면, 좌측, 우측, 상방)
Step 3: 세팔로메트릭 측정 (자동)
Step 4: EZM 진단 로직 실행
Step 5: 치료 계획 생성
Step 6: 결과 보고서 생성
