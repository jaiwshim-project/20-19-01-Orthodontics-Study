## 📊 각 팀별 상세 계획

### 🔵 **팀 1: 퀴즈 시스템**

#### Phase 1 퀴즈 (10문제)
```
1. EZM이란?
2. 정상교합 특징
3. 부정교합 분류
4. 성장 패턴
5. 진단 기초
6. (사진 보기) 이 케이스는?
7. (선택형) 다음 중...
8-10. 개념 적용 문제
```

#### Phase 2 퀴즈 (15문제)
```
1-10: 공간분석, 발치 판단, 진단
11-15: 케이스 분석 (주관식)
```

#### Phase 3 퀴즈 (5개 케이스)
```
- 각 케이스별 진단 및 치료 계획 작성
- 자동 평가 (체크리스트 기반)
```

---

### 🟠 **팀 2: 케이스 스터디**

#### 케이스 항목 (각 1개)
```
✓ Before 사진 (정면, 측면, 상하악)
✓ 환자 정보 (나이, 성별, 주소증)
✓ 진단 (EZM 적용)
✓ 치료 계획 (발치 여부, 장치, 기간)
✓ After 사진
✓ 결과 분석
✓ 학습 포인트
```

#### 추천 케이스 주제
```
1-5: 정상교합 → 경미한 부정교합 (초급)
6-10: 중등도 부정교합 (중급)
11-15: 복잡한 부정교합 (고급)
16-20: 실패 케이스 & 개선 (고급+)
```

---

### 🟡 **팀 3: 세팔로 도구**

#### 기능 명세
```
1. 이미지 업로드
   - JPG/PNG 지원
   - 최대 5MB
   - 캐시 저장

2. 기준점 표시 (Canvas)
   - 클릭으로 점 추가
   - 자동 번호 지정
   - 수정/삭제 가능

3. 측정값 계산
   - 거리: 2점 간 거리
   - 각도: 3점으로 각도 계산
   - 비율: 거리 비율

4. 결과 저장
   - JSON으로 저장
   - 이미지 + 측정값 함께
   - 비교 기능
```

#### 기본 기준점 (8개)
```
1. Na (Nasion) - 비점
2. A point - 상악골점
3. B point - 하악골점
4. Pog (Pogonion) - 이각점
5. Me (Menton) - 하악골 최하점
6. PNS (Posterior Nasal Spine)
7. Ba (Basion)
8. S (Sella)
```
